About

Emoji Weather is a website that allows a user to search, text, or call to receive the weather conditions.

What is the point of this?

I made it for fun.

How does it work?

A user texts or calls a phone number that is registered with Twilio. Twilio transforms the text or voice data into an HTTP POST request and sends it to a publicly accessible URL of the website.

The body of the request is converted from a messy search query into a proper geographic location with Requests and Google’s Geocoding API. The location’s latitude and longitude is sent to the Dark Sky API, which returns the weather conditions. The website generates a TwiML response for Twilio, which then transforms it into a text message for the user. Searching is handled similarly but makes requests directly to Google and Dark Sky.

The website was written in the Python programming language with the Django web framework. The front-end code was written with the Bootstrap library. The code is available on GitHub. The website is hosted on Heroku.
